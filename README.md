# Student Management System

A full stack web application built using:

- Node.js
- Express.js
- MongoDB
- JWT Authentication
- EJS

## Features

- User Registration
- User Login with JWT
- Protected Dashboard
- Add Students
- View Students
- Logout
- MongoDB Database Integration

## Installation

1. Clone the repository
2. Run:

npm install

3. Create a .env file:

PORT=3000
MONGO_URI=mongodb://127.0.0.1:27017/cognifyDB
JWT_SECRET=your_secret_key

4. Start server:

node server.js

Open:
http://localhost:3000
